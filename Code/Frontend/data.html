Saved data

Here you can request previous data tracked by the project. Simply fill in an amount of measurements you want to see, and optionally a start and end time, then press submit.

Leaving the startdate blank will give you the requested amount before the enddate. By default the enddate is set to the most recent measurement, so leave this blank to get the latest values.
